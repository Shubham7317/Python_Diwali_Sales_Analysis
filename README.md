🪔 Python Diwali Sales Analysis

📌 Project Overview
This project focuses on analyzing Diwali sales data using Python to uncover customer purchasing behavior, sales trends, and business insights. The analysis helps businesses understand which products, categories, and customer segments perform best during the festive season.

🚀 Features
📊 Data cleaning and preprocessing
📈 Exploratory Data Analysis (EDA)
👥 Customer segmentation (age, gender, occupation)
🛍️ Product category analysis
💰 Revenue and sales insights
📉 Visualization using charts and graphs

📊 Key Insights
Female customers contribute more to overall sales
Most buyers belong to the 26–35 age group
Married customers have higher purchasing power
Top-performing categories include clothing and electronics
Certain states contribute significantly to total revenue

🛠️ Tools & Technologies
Python
Pandas – Data manipulation
NumPy – Numerical operations
Matplotlib & Seaborn – Data visualization
Jupyter Notebook – Analysis environment
